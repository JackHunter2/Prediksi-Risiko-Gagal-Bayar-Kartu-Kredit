🎯 FITUR INI UNTUK APA?
───────────────────────────────────────────────────────────────
Fitur ini memungkinkan Anda mengisi form secara OTOMATIS dengan
data contoh untuk keperluan:
  • Demo ke klien/dosen
  • Testing cepat
  • Pelatihan user baru
  • Pemahaman pola data

⚡ CARA PAKAI (SUPER MUDAH!)
───────────────────────────────────────────────────────────────

Metode 1: TOMBOL
1. Lihat tombol 📋 di KANAN BAWAH layar (warna orange)
2. Klik tombol tersebut
3. Pilih contoh yang diinginkan:
   • ✅ Risiko Rendah (nasabah baik)
   • ⚠️ Risiko Tinggi (nasabah bermasalah)
4. Form terisi OTOMATIS dalam 2 detik!

Metode 2: KEYBOARD
1. Tekan tombol F2 di keyboard
2. Pilih contoh
3. Selesai!

🚀 CARA PAKAI UNTUK DEMO
───────────────────────────────────────────────────────────────

SKENARIO: Demo ke Dosen/Klien

1. Buka aplikasi
   "Ini adalah sistem prediksi risiko kredit..."

2. Klik tombol 📋
   "Untuk demo, saya akan gunakan contoh data..."

3. Pilih "Risiko Rendah"
   "Ini contoh nasabah dengan profil baik..."
   (Form terisi otomatis dengan animasi keren!)

4. Submit
   "Dan hasilnya... Risiko Rendah dengan probabilitas X%"
   (Tunjukkan hasil + rekomendasi)

5. Kembali, klik 📋 lagi

6. Pilih "Risiko Tinggi"
   "Sekarang kita coba nasabah bermasalah..."
   (Form terisi otomatis lagi)

7. Submit
   "Dan hasilnya... Risiko Tinggi dengan probabilitas Y%"
   "Lihat perbedaannya?"

Total waktu: < 2 menit untuk demo lengkap!

───────────────────────────────────────────────────────────────

   → Jelaskan sambil form terisi otomatis
